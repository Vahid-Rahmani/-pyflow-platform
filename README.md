# Python Learning Platform (PyFlow)

PyFlow is a modern, interactive, and AI-powered education platform designed to transform beginners into proficient Python and AI developers. Built with a "Learn-by-Doing" philosophy.

## 🚀 Features
- Interactive Coding Lessons: Short, gamified, and engaging content.
- AI-Powered Tutor: Real-time debugging, personalized explanations, and coding mentorship.
- Integrated Code Editor: Secure, sandbox-based execution environment (Monaco Editor).
- Multi-Language Support: Fully localized for German, English, Turkish, Russian, and Arabic.
- PWA Experience: Optimized for mobile and desktop, feeling like a native application.
- Gamification: XP system, streaks, and achievements to keep users motivated.

## 🛠 Tech Stack
- Backend: Python (Django, Django REST Framework)
- Frontend: React, Tailwind CSS
- Database: PostgreSQL
- Infrastructure: Docker, Redis, Celery (for task execution)
- AI Integration: OpenAI API / Custom LLM Agents

## 🏗 Project Architecture
The project follows a modular design, separating the core application into distinct components:
- users/: Custom authentication and user profile management.
- courses/: Content management system (CMS) with i18n support.
- sandbox/: Secure code execution environment for Python.
